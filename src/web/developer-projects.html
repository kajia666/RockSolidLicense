Workspace Notes

Creating a new project is owner-only. The owner account automatically becomes the project owner developer.

Project profile editing, status switching, feature toggles, and SDK credential rotation require product write permission. In the current role model that means the owner account or an assigned `admin` member.

The detail panel merges project settings with the scoped dashboard metrics already exposed by the developer summary API.
