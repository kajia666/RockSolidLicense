Export Notes

The integration package is generated from the currently selected scoped project and always reflects the latest SDK credentials visible to the current developer actor.

After rotating `sdkAppSecret` or `sdkAppId`, refresh the package here and redeploy the client-side configuration before shipping a new build.

The generated HTTP host and base URL come from the current request origin. Replace them with your public domain or TLS endpoint when deploying behind a reverse proxy.
