RockSolidLicense Console

面向商业软件授权的运营控制台。这里可以创建产品、授权策略和卡密批次，也可以直接查看用户账号、 在线会话、设备绑定与审计日志，并执行账号禁用、会话下线、设备释放等后台操作。

默认管理员：admin / ChangeMe!123
HTTP 管理入口：/api/admin/*
客户端授权入口：/api/client/* + TCP
尚未登录管理后台。

管理员登录

登录成功后，下面所有管理接口都会自动带上 Bearer Token。

创建产品

创建策略

支持字段：deviceFingerprint、machineCode、machineGuid、cpuId、diskSerial、boardSerial、biosSerial、macAddress、installationId、requestIp、publicIp、localIp。

策略运行配置

作者可以随时切换是否允许多开，以及决定换绑检测是否参考硬件信息和请求 IP。

批量生成卡密

创建策略成功后，返回结果里会携带策略 ID。生成完成后可在输出面板中复制卡密。

Entitlement Control

Resume or freeze an entitlement, extend duration licenses, or top up point-based access without leaving the admin console.

审计快捷筛选
点击下方审计日志行，会自动回填事件筛选、产品代号、用户名以及相关操作 ID。
支持通过 /api/admin/ops/export 和 /api/admin/ops/export/download 预览或下载管理员授权运营快照包。
先点击 Preview Ops Snapshot，生成一份可交接的授权运营快照概览。

账号控制

禁用账号会立即踢下该账号下所有在线会话；恢复为 active 后允许重新登录。

会话控制

会话被下线后，旧的 sessionToken 心跳会立即失效。

设备控制

释放设备会腾出设备名额，并使该设备当前在线会话失效。之后该设备可重新绑定。

设备封禁

封禁设备会立即踢下相关在线会话，并阻止该设备再次登录。解除封禁后，该设备可以重新登录和绑定。

版本状态控制

你可以把旧版本直接置为 disabled，也可以把活跃版本标记为 force update，用于强制升级。

操作提示

控制台就绪，等待你的操作。
小技巧：点击下方表格中的任意一行，会自动把对应的账号 ID、会话 ID 或绑定 ID 带到上面的操作输入框。
